# docs

Place a screenshot of the running dashboard here as **`dashboard.png`**.

How to capture it:

1. Run the app and open http://localhost:8081/dashboard.html
2. Sign in (demo@pulseguard.dev / password123) and wait ~1 minute for the monitors to turn green.
3. Take a screenshot and save it as `docs/dashboard.png`.

The main `README.md` references `docs/dashboard.png`, so once the file is here it will
render automatically on GitHub.
